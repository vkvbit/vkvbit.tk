Introduction:

Folloowing things have been used to create TrackURL:-

  • Bash Bash is used to create a script which will automate the process in Terminal.
  • ngrok ngrok is used here to host a webpage which will helps us to locate the target's device.
  • HTML HTML is used to create a webpage containing a program that access the location of website.
  • CSS CSS is used here to make the webpage attractive.
  • JavaScript JavaScript is used to create the program which will access the location of target's device.

TrackURL provides accurate location of target because TrackURL access the GPS location of target's device and send back it to the admin's Terminal.

Usage:

Clone It!

First, You need to clone the TrackUrl from github. To do that, you need to type this command :

For Gitlab users-

git clone https://gitlab.com/vkvbit/TrackURL.git

For Github users-

git clone https://github.com/vkvbit/TrackURL.git

Then, You will need to cd and chmod it.

Edit It (Optional - only for 64-bit)

Then, you will need to edit the TrackUrl program.
To edit it, write : nano TrackUrl.sh

If you have a 32-bit linux, then you don't need to edit anything, it is set as default, but if you have the 64-bit, then you have to edit it.

Starting the Program

Well, after you done that, press Ctrl+X, then press y, then Enter.
Now, you have to get in the TrackUrl, Just by typing :

./TrackURL.sh

Then, you will realize there are two terminals. from the X-term terminal, copy the link, and paste it in the main terminal.

Copy the one with "https" in it, then press enter.

Victim's Location

Now, you need a bit of social engineering, you should send the link to the victim and let the victim trust it is not a malicious website.
Now, wait for victim to open it.

Now, in your tail terminal, there will be bunch of sentences. Select the number which is at the bottom, and which after "/logme/" and behind "HTTP/1.1/".

Now, go to Google Map and paste or search the co-ordinates.

You have the victim's Location.
